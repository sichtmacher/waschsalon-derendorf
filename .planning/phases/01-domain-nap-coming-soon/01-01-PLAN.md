<task type="auto">
  <name>Task 1: Scaffold Astro 6 project and configure stack</name>
  <files>package.json, astro.config.mjs, netlify.toml, src/styles/global.css, public/robots.txt, public/favicon.svg</files>
  <action>
1. Initialize git repo in project root: `git init`

2. Scaffold Astro 6 project in current directory (not a subdirectory). Run:
   ```
   npm create astro@latest . -- --template minimal --install --no-git --typescript strict
   ```
   Use the minimal template to keep it clean. The `--no-git` flag avoids re-initializing git.

3. Install additional dependencies:
   ```
   npm install tailwindcss @tailwindcss/vite @astrojs/sitemap
   ```

4. Configure `astro.config.mjs` with:
   - `site: "https://waschsalon-derendorf.de"`
   - Tailwind v4 via Vite plugin: `vite: { plugins: [tailwindcss()] }` (import from `@tailwindcss/vite`)
   - Astro 6 Fonts API: `fonts: [{ provider: fontProviders.google(), name: "Inter", cssVariable: "--font-body" }]` (import `fontProviders` from `astro/config`)
   - Sitemap integration: `integrations: [sitemap()]` (import from `@astrojs/sitemap`)

5. Create `src/styles/global.css` with Tailwind v4 CSS-first config:
   ```css
   @import "tailwindcss";

   @theme {
     --color-brand: #1D4ED8;
     --color-brand-light: #E0F2FE;
     --color-surface: #F3F4F6;
     --color-text: #1F2937;
     --font-body: var(--font-body);
   }
   ```

6. Create `netlify.toml`:
   ```toml
   [build]
     command = "npm run build"
     publish = "dist"

   [build.environment]
     NODE_VERSION = "22"
   ```

7. Create `public/robots.txt`:
   ```
   User-agent: *
   Allow: /

   Sitemap: https://waschsalon-derendorf.de/sitemap-index.xml
   ```

8. Create `public/favicon.svg` with the basket emoji:
   ```xml
   <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
     <text y=".9em" font-size="90">&#x1F9FA;</text>
   </svg>
   ```

9. Remove the default `src/pages/index.astro` that the template creates (Plan 02 will create the real one).

10. Create a `.gitignore` if the scaffold did not (ensure `node_modules/`, `dist/`, `.astro/` are ignored).

IMPORTANT: Do NOT create a `tailwind.config.js` -- Tailwind v4 uses CSS-first configuration only. Do NOT install `@astrojs/tailwind` -- it is deprecated; use `@tailwindcss/vite` directly.
  </action>
